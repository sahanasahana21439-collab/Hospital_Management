# Database — Neon PostgreSQL

## Overview
This folder contains all database-related files for the Hospital Management System, hosted on **Neon PostgreSQL**.

## Files
| File | Purpose |
|---|---|
| `schema.sql` | Database schema (tables, indexes, constraints) |

## Setup
1. Create a Neon project at [neon.tech](https://neon.tech)
2. Use the connection string from the Neon dashboard
3. Run `schema.sql` against your Neon database to create tables

## Connection
The Neon API token is stored in the project root `tokens` file.
